CSS Display: Grid

CSS has a built in grid system that can be used by setting the display attribute to grid. It works well when the grid pattern is regular, for instance 4 rows of 3 columns each. Implementation is more difficult in instances where each row has a different number of columns.

The CSS Grid system does allow for nested grids, i.e. grids within grids. This aids the flexibility of the system.
