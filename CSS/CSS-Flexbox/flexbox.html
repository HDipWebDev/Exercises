Introduction to Flexbox

A different approach to layout is used for flexbox. In this instance we do not specify a container width; we set the width to 'auto' and let flexbox change the column widths as necessary

Media queries are still necessary. We use these to control how the divs are ordered on the page. I have used the media queries to change the left and right margins, and to stack the divs upto a max-width of 481px
