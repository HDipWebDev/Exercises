Introduction

HTML5 made some major improvements to form functionality. Before HTML5, a considerable amount of effort was required to validate form information. HTML5 expanded the standard <input> tag to allow for greater control of what is entered. Input types that you can specify now include:

  • text
  • email
  • search
  • url
  • number
  • tel
  • range
  • date
  • month
  • week
  • time
  • datetime
  • color

HTML5 Improvements

In addition to the new input types, HTML5 introduced some interesting attributes for form an input behaviour. Previously this functionality required considerible work in JavaScript. The new attributes now include:

  • required user must enter a value
  • placeholder displays a prompt in the input field
  • autofocus forces the input cursor to input field
  • autocomplete when set to on, will turn on auto complete functionalty on browser
  • formnovalidate turns off form validation (generally not a good idea)
  • multiple user can enter multiple values
