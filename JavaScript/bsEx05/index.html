HTML5 Data Storage

HTML5 introduced local storage as a potential replacement for cookies. Cookies are limited in size to 4k. Local storage allows for more storage capacity. W3C guidelines indicate that local storage should be capped at 5Mb. Storage/Retrieval is achieved by means of a key-value pair. The value is limited to string values. This is considered by many as a severe limitation of the system, in particular in relation to objects. However objects can be stored using JSON.stringify(object).

localStorage Object

The localStorage object has the following methods.

Method Discription
localStorage.get(key) Get the value associated with key
localStorage.setItem(key, value) Create a new key, value pair
localStorage.removeItem(key) Remove the item associated with key.
