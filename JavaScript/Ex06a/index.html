LightBox Modal

In this example, images are placed in col-3 divs below, and when clicked are launched in a modal. The process is quite simple:

  • Each thumbnail is given a special class named t-nail. This class is used to attach a click listener to each image.
  • When the image is clicked, the image, as this, is passed to the handler.
  • The handler grabs the alt text, and uses it for the modal title. This is perhaps not the most robust approach..
  • The handler also grabs the src value and sets the modal image source to same.
  • Close button functionality is handled by Bootstrap.
