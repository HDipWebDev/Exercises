Web SQL

WebSQL is a lightweight, but fully functionalm DBMS intended for the storage of small amounts of relational data.

In the time honored tradition of disagreement, some big names in Web disagreed on how to implement browser side storage. WebSQL supported by Google and apple, lost out to IndexedDB proposed by Mozilla and Microsoft.
