The object of this exercise is to demonstrate how the this keyword and functions can help us to write efficent code. The example provided here runs the Modernizr script to help with backwards compatibility. Modernizr is a library that includes HTML5Shiv and additional support for CSS3. Please refer to http://modernizr.com/ to get the latest version.

The this keyword will not work in IE8 or below. Quite a bit more work is required for full backwards compatibility.
