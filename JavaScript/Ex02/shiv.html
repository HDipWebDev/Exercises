The object of this exercise is to demonstrate how the this keyword and functions can help us to write efficent code. The example provided here runs HTML5Shiv to help with backwards compatibility. HTML5Shiv is a script to allows non-HTML5 browsers to interpret HTML5. In effect, what it does is provide support for nav, section, article etc. Please refer to https://code.google.com/p/html5shiv/ to get the latest release.
