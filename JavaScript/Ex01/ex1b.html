Creating Better Events

Part of the problem with the previous event handler is that it involves placing bits of JavaScript into the site HTML. This creates unnecessary complexity in the code and makes maintenance extremly difficult.

A better implementation is to keep the event listeners in JavaScript by using document.getElementById("someID").onclick.

When the event handler is added in, the final code looks like: document.getElementById("someID").onclick = doSomething();.
